# Paper draft

`paper.tex` is the IEEE-conference draft. To build:

```cmd
cd research_v2\paper
pdflatex paper.tex
bibtex   paper
pdflatex paper.tex
pdflatex paper.tex
```

Figures are pulled from `../figures/` via `\graphicspath`. Tables are
generated by `python -m research_v2.src.paper.latex_tables` after
Stage 1 / Stage 2 results are present.

See also:
- `METHODOLOGY.md`        — full methodology in prose
- `METHODOLOGY_SIMPLE.md` — four-box simplified version
- `methodology_combined.docx` — methodology DOCX (rebuild with
  `python -m research_v2.src.paper.build_methodology_docx`)
